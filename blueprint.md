# Blueprint for Geopoint AI

## Overview

A web application that leverages AI to provide geopoint-related services. The application features a modern, visually appealing design with a focus on user experience and accessibility.

## Project Structure

*   `index.html`: The main entry point of the application.
*   `style.css`: The stylesheet for the application.
*   `main.js`: The main JavaScript file for the application, including web components and interactive features.
*   `blueprint.md`: This file, which documents the project.

## Features

*   **Modern HTML Structure:** A clean and semantic HTML5 structure.
*   **Modern CSS:** A responsive and visually appealing design using modern CSS features.
*   **Web Component:** A simple "hello world" web component to demonstrate the use of modern JavaScript.
*   **Legal Modal:** A modal for Privacy, Terms, and Cookies with an expandable FAQ-style layout.
*   **Firebase Hosting:** The application will be deployed to Firebase Hosting.
